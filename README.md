# COMP4537-AI_Project

## Set up
```
`cd Frontend`
`npm install react-router-dom axios react-bootstrap bootstrap`
`npm install react react-dom react-scripts web-vitals`
`npm start`
```
```
`cd Backend`
`npm init -y`
`npm install express mongoose dotenv bcryptjs jsonwebtoken cookie-parser cors axios`
`node server.js`
```
